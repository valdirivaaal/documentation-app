### 1. Menu Pendataan

##### 1.1. Jenis Tabungan
Jenis Tabungan adalah fitur untuk menambahkan jenis tabungan.

Langkah - langkah untuk menambahkan data jenis tabungan :
1. Pilih menu Pendataan - Jenis Tabungan

    <img src="../images/1_Menu-Pendataan.png">

2. Klik tombol Add di pojok kanan atas tabel.

    <img src="../images/1_A_Tampilan_Table_Jenis_Tabungan.png" alt="drawing" width="550" />

3. Isilah data pada form yang muncul.

    <img src="../images/1_A_Tampilan_Add_isian_Jenis_Tabungan.png" alt="drawing" width="550" />

4. Setelah itu klik Simpan, data yang diinput akan tampil pada halaman tabel Jenis Tabungan

Langkah - langkah untuk meng-edit data jenis tabungan :
1. Pilih menu Pendataan - Jenis Tabungan.

    <img src="../images/1_Menu-Pendataan.png">

2. Klik tombol Edit pada data yang ingin diubah.

    <img src="../images/1_A_Tampilan_Table_Jenis_Tabungan.png" alt="drawing" width="550" />

3. Data yang dipilih tadi, akan muncul dalam tampilan form.

    <img src="../images/1_A_Tampilan_Edit_Jenis_Tabungan.png" alt="drawing" width="550" />

4. Setelah selesai meng-edit, klik tombol Simpan.


Langkah - langkah untuk menghapus data jenis tabungan :
1. Pilih menu Pendataan - Jenis Tabungan.

    <img src="../images/1_Menu-Pendataan.png">

2. Klik tombol Delete pada data yang ingin dihapus.

    <img src="../images/1_A_Tampilan_Table_Jenis_Tabungan.png" alt="drawing" width="550" />

3. Setelah muncul *pop-up* konfirmasi, pilih Yes.

    <img src="../images/1_A_Tampilan_Delete_Jenis_Tabungan.png" alt="drawing" width="550" />

4. Data terhapus.
